# Care Plus - Front-End Project

## Overview

This project was developed for **Care Plus** as part of the **Challenge Project** for the second semester of the **Analysis and Systems Development** program at **FIAP**.

The objective of this project is to design and implement front-end interfaces that simulate a real-world digital solution for the company. The focus is on building responsive and user-friendly screens using modern web development technologies.

## Technologies Used

The front-end of this project was developed using the following technologies:

```
Frontend:
├── HTML5  
├── CSS3  
├── Bootstrap 5  
└── JavaScript (ES6+)
```

These technologies were chosen to ensure responsive layouts, visual consistency, and interactive user interfaces.

## Project Structure

The project consists of multiple front-end screens designed to represent different parts of the application. The interfaces were developed following best practices for layout organization, responsiveness, and maintainability.
